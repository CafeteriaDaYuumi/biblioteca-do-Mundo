# RPG Steampunk Wiki

> **Versão:** 0.0 - Arquitetura e Documentação Base

## Sobre o projeto

O **RPG Steampunk Wiki** é um projeto dedicado à criação de uma plataforma de documentação para um universo original de RPG de mesa ambientado em um cenário steampunk.

O objetivo da Wiki é servir como uma enciclopédia completa do universo, reunindo todas as informações necessárias para jogadores, mestres e desenvolvedores. O sistema foi concebido para centralizar o conhecimento do mundo de forma organizada, facilitando consultas, expansão do cenário e manutenção da consistência entre todos os elementos da narrativa.

Diferente de um site tradicional, esta aplicação foi planejada desde o início para funcionar como uma base de conhecimento, permitindo que o universo cresça continuamente sem comprometer sua organização.

---

# Objetivos

O projeto possui como principais objetivos:

* Documentar completamente o universo do RPG.
* Organizar o conteúdo em módulos independentes.
* Facilitar a navegação entre informações relacionadas.
* Manter uma estrutura escalável para centenas ou milhares de páginas.
* Servir como referência oficial do cenário.
* Possibilitar futuras expansões sem necessidade de reformulação da arquitetura.

---

# Estado atual

Atualmente o projeto encontra-se na fase inicial de desenvolvimento.

Nesta etapa, o foco não está na implementação visual ou funcional da aplicação, mas na definição da arquitetura que sustentará todo o projeto futuramente.

Esta versão contempla:

* definição da estrutura de diretórios;
* organização dos módulos da Wiki;
* documentação da arquitetura;
* planejamento das funcionalidades;
* padronização dos nomes das páginas;
* definição da estrutura de navegação.

Todo o conteúdo desta fase é voltado para documentação e planejamento.

---

# Filosofia do projeto

O universo do RPG será desenvolvido primeiro em sua totalidade, para somente depois receber sua implementação definitiva dentro da Wiki.

Essa abordagem permite que toda a estrutura do cenário seja construída de forma consistente antes da produção do conteúdo final.

O desenvolvimento seguirá as seguintes etapas:

1. Planejamento da arquitetura.
2. Documentação do universo.
3. Implementação da Wiki.
4. Inserção do conteúdo definitivo.
5. Internacionalização.

---

# Organização do projeto

A Wiki será organizada por grandes módulos temáticos.

Cada módulo será responsável por documentar uma área específica do universo.

Exemplos:

* Mundo
* Personagens
* História
* Facções
* Localizações
* Tecnologia
* Equipamentos
* Criaturas
* Sistema de RPG

---

# Tecnologias

A primeira versão será construída utilizando tecnologias web nativas.

* HTML5
* CSS3
* JavaScript

Conforme o projeto evoluir, novas tecnologias poderão ser incorporadas para melhorar desempenho, manutenção e escalabilidade.

---

# Idiomas

O desenvolvimento será realizado em duas etapas.

### Primeira etapa

Toda a documentação será produzida em português (PT-BR).

### Segunda etapa

Após a conclusão da documentação e estabilização da arquitetura, será iniciado o processo de internacionalização para inglês (EN), mantendo ambas as versões sincronizadas.
